# Originary brand guide

## Canonical direction
- Wordmark: `originary`
- Case: lowercase
- Tone: calm, premium, infrastructural, minimal
- Color: `#0B0B0C` on light backgrounds
- Reverse color: `#FFFFFF` on dark backgrounds

## Recreation typeface
- Preferred source typeface for recreation: Inter Display Medium
- Use the supplied outlined SVGs for brand-critical work
- Do not substitute random fallback fonts for the canonical logo

## Philosophy
The logo should feel:
- minimal
- premium
- restrained
- contemporary
- calm
- trustworthy

It should not feel:
- gimmicky
- symbolic
- playful
- over-designed
- trendy in an obvious way

## Usage
- Use `originary-canonical.svg` as the primary master
- Use `originary-reverse.svg` on dark surfaces
- Use `originary-compact.svg` only in constrained horizontal placements
- Do not stretch, skew, shadow, bevel, outline, or recolor the logo outside the monochrome system
